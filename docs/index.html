公众号内容提取工具页

这是一个纯静态的 GitHub Pages 工具页。它可以直接加载示例数据、导入你本地生成的 article.json、或者通过 URL 参数读取远程 JSON。 如果要在网页里直接粘公众号链接并自动抓取原文，需要额外的后端服务，不适合只靠 GitHub Pages。

直接使用

你现在打开这个页面就能用，不需要本地环境。支持三种方式：默认示例、上传本地 JSON、远程 JSON 地址。

适用范围

适合展示、归档、分享和二次利用提取结果。你可以把脚本生成的 article.json 直接丢进来，页面会自动解析正文、图片和视频。

怎么接上你自己的结果

1. 用提取脚本生成 `article.json`。
2. 打开这个网页。
3. 选择“上传 JSON”，立即查看。
4. 如果你把 JSON 放到公网，再选择“远程 JSON”，就能把这个工具页当成通用查看器。

提示：分享链接会把远程 JSON 地址写进 `?data=` 参数里，方便直接转发。
