WeChat Link Extractor

公众号文章提取在线工具

这个页面现在支持真正的“输入公众号链接 -> 自动提取”。 实现方式是 GitHub Pages 前端 + 一个在线 API 后端； 页面仍然可以继续加载示例数据、上传本地 article.json、或者读取远程 JSON。

四种使用方式

默认示例、在线提取、上传 JSON、远程 JSON。你可以把这个页面当作在线查看器，也可以直接当成提取入口。

在线提取说明

在线提取会把你输入的 mp.weixin.qq.com 链接发送到后端， 后端使用真实浏览器抓取正文、图片和视频，再把结构化结果返回给这个页面展示。
